# Site-livre
Inicio ao PHP e estilização do site.
Este site foi desenvolvido como parte de uma tarefa de aula, com o objetivo de praticar PHP e técnicas de estilização web. O tema escolhido foi "Harry Potter", uma paixão pessoal que inspirou o design e o conteúdo do projeto. A página combina funcionalidades dinâmicas em PHP com elementos visuais que remetem ao universo mágico da série.
